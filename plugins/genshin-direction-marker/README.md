# 原神方向标记插件

识别攻略视频字幕中的方位词，在原神小地图位置显示方向标记。

## 功能

- 自动识别字幕中的方向词（东/西/南/北、东北/东南/西北/西南）
- 支持小地图方向表达（右上/右下/左上/左下）
- 支持钟点方向（3点钟方向、12点方向等）
- 在小地图位置显示对应的方向箭头

## 安装

1. 将插件文件夹复制到 `User/Data/Profiles/genshin/plugins/direction-marker/`
2. 重启应用或重新加载 Profile

## 配置

在 `config.json` 中可调整以下参数：

```json
{
  "overlay": {
    "x": 43,
    "y": 43,
    "size": 212
  },
  "markerDuration": 0
}
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| overlay.x | 覆盖层 X 坐标 | 43 |
| overlay.y | 覆盖层 Y 坐标 | 43 |
| overlay.size | 覆盖层大小 | 212 |
| markerDuration | 标记显示时长（毫秒），0 为常驻 | 0 |

## 支持的方向表达

### 基本方向
- 东、西、南、北
- 东北、东南、西北、西南
- 东方向、西边、南方、北侧 等

### 小地图方向
- 地图右上、地图左下
- 小地图右、小地图左
- 地图上方、地图下方

### 钟点方向
- 3点钟方向 → 东
- 12点方向 → 北
- 9点钟 → 西

## 权限要求

- `subtitle` - 读取视频字幕
- `overlay` - 显示覆盖层
